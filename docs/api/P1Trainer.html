P1Trainer¶

module trainer.P1Trainer¶
class P1Trainer¶

The Precision-Level 1 (P1) Trainer class object instantiates a training session using PyTorch for taking an input dataset and selected pretrained model from PyTorch model zoo in order to generate a custom-trained P1 ONNX model file.

This model can then be deployed as a ROS2 package.
